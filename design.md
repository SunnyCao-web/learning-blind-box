# 学霸盲盒 (Learning Blind Box) - 移动应用设计文档

## 应用概述

**学霸盲盒**是一款为青少年设计的学习激励应用，通过将学习任务与盲盒收集机制结合，帮助学生提升专注力、克服拖延症，并在学习中获得持续的成就感。

---

## 主要内容与功能 (Primary Content and Functionality)

### 1. 欢迎屏幕 (Welcome Screen)

**布局：**
- 背景：深蓝色星空，营造沉浸感
- 中央：应用名称"学霸盲盒"（艺术字）
- 下方："出发"按钮

**功能：**
- 点击"出发"进入主题选择

---

### 2. 主题选择 (Theme Selection)

**布局：**
- 顶部：标题"请选择你的宝藏主题"
- 中央：3-4个可滑动的主题卡片
  - 鬼灭之刃（带"灭"字背景）
  - 航海王（带骷髅旗背景）
  - 自定义主题（问号+画笔图标）
- 底部：确认按钮"就选它了！"

**功能：**
- 选择预设主题或自定义
- 确认后进入主界面

---

### 3. 主界面 (Home Screen)

**布局：**
- 顶部横幅：当前主题logo + 进度显示（已收集 X/9）
- 中央：巨大的"+"按钮（邀请创建任务）
- 底部导航栏：主页、图鉴、设置

**功能：**
- 点击"+"创建新任务
- 点击进度条进入图鉴
- 显示待领取的"侦探奖励"（如有）

---

### 4. 任务创建 (Task Creation)

**布局：**
- 弹出卡片式对话框
- 输入框：任务描述（如"完成5道数学题"）
- 时长选择：15分钟、25分钟、45分钟按钮
- 底部：确认"开始专注"按钮

**功能：**
- 输入任务内容
- 选择专注时长
- 启动计时器

---

### 5. 专注计时 (Focus Timer)

**布局：**
- 全屏沉浸界面
- 背景：缓慢变化的星空或深海动画
- 中央：巨大的倒计时数字（如 25:00）
- 底部：长按"放弃"按钮（防止误触）

**功能：**
- 倒计时自动运行
- 屏蔽通知和干扰
- 时间到自动跳转到成果判定

---

### 6. 成果判定 (Judgment Screen)

**布局：**
- 顶部：标题"恭喜完成专注！请判定你的成果"
- 中央：两个大按钮
  - [宝箱图标] **我解开了谜题！** (做对了)
  - [放大镜图标] **开启侦探模式** (做错了)

**功能：**
- 用户自主判定成果
- 点击任一按钮进入相应流程

---

### 7. 开盲盒 (Blind Box Opening)

**布局：**
- 屏幕变暗，聚光灯照亮中央宝箱
- 华丽的日式宝箱，轻微晃动
- 提示文字："点击开启"

**动画流程：**
1. 用户点击宝箱
2. 宝箱盖子"砰"地弹开
3. 金光喷射充满屏幕
4. 光芒散去，展示随机解锁的卡片

**卡片展示：**
- 立体旋转的卡片
- 角色图片（如"灶门炭治郎"）
- 角色名称 + 稀有度标记（R/SR/SSR）
- 底部："收入图鉴"按钮

---

### 8. 收藏图鉴 (Collection Gallery)

**布局：**
- 3x3网格布局
- 已解锁：全彩卡片
- 未解锁：灰色剪影 + 问号

**功能：**
- 展示所有可解锁的卡片
- 点击卡片查看详情（可选）
- 显示总进度（已收集 X/9）

---

## 关键用户流程 (Key User Flows)

### 流程 A：首次启动 → 选择主题 → 进入主界面

1. 打开App → 欢迎屏幕
2. 点击"出发" → 主题选择屏幕
3. 选择"鬼灭之刃" → 点击"就选它了！"
4. 进入主界面（显示"鬼灭之刃 0/9"）

### 流程 B：完成任务 → 开盲盒（成功路径）

1. 主界面点击"+" → 任务创建
2. 输入"完成5道数学题"，选择"25分钟" → 点击"开始专注"
3. 进入计时器，倒计时至 00:00
4. 自动进入成果判定 → 点击"我解开了谜题！"
5. 进入开盲盒界面 → 点击宝箱 → 动画播放
6. 获得"灶门炭治郎"卡片 → 点击"收入图鉴"
7. 返回主界面，进度更新为"1/9"

### 流程 C：完成任务 → 侦探模式（学习路径）

1. 完成任务后进入成果判定
2. 点击"开启侦探模式"
3. App提示"请先完成探案报告，再来领取奖励哦！"
4. 用户稍后点击"领取侦探奖励" → 开盲盒
5. 获得奖励卡片

## 配色方案 (Color Choices)

| 用途 | 颜色 | 十六进制 | 说明 |
|------|------|---------|------|
| **主色** | 深蓝 | #0a7ea4 | 专业、沉着、适合学习 |
| **背景** | 白色/深灰 | #ffffff / #151718 | 浅色/深色模式 |
| **表面** | 浅灰 | #f5f5f5 | 卡片、按钮背景 |
| **文字** | 深灰/白色 | #11181C / #ECEDEE | 主文本 |
| **辅助文字** | 中灰 | #687076 | 次要信息 |
| **成功** | 绿色 | #22C55E | 完成、正确 |
| **警告** | 橙色 | #F59E0B | 提醒、侦探模式 |
| **错误** | 红色 | #EF4444 | 错误、失败状态 |

**品牌特色：**
- 采用现代、简洁的设计风格
- 避免过于幼稚的元素
- 强调"游戏化"但不失"学习"的严肃感
- 深蓝色营造专注的氛围

## 交互特性

### 动画与反馈
- **按钮按压：** 缩放 0.97 + 轻微触觉反馈
- **页面切换：** 淡入淡出（200-300ms）
- **开盲盒：** 华丽的金光特效（800ms）
- **卡片旋转：** 立体旋转动画（600ms）

### 无障碍设计
- 所有按钮都有清晰的标签
- 支持系统字体大小调整
- 高对比度文本
- 支持深色模式

---

## 技术实现细节

### 状态管理
- 使用 React Context + AsyncStorage 管理本地状态
- 存储项目：当前主题、已解锁卡片、任务历史

### 本地存储结构
```json
{
  "currentTheme": "demon-slayer",
  "themes": {
    "demon-slayer": {
      "name": "鬼灭之刃",
      "unlockedCards": [0, 3, 5],
      "totalCards": 9
    }
  },
  "taskHistory": [
    {
      "date": "2026-03-10",
      "taskName": "完成5道数学题",
      "duration": 25,
      "result": "success"
    }
  ]
}
```

### 盲盒卡片数据
- 预设主题（鬼灭之刃等）包含 9 张卡片
- 自定义主题由用户上传图片组成
- 每次开盒随机抽取未解锁的卡片
